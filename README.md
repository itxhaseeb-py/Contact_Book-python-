# 📒 Python Contact Book

A simple command-line Contact Book application built with Python. This project allows users to store and manage contact information efficiently using a text file as a lightweight database.

---

## 🚀 Features

* ➕ Add new contacts
* 🔍 Search contacts
* ✏️ Update existing contacts
* 🗑️ Delete contacts
* 📋 View all saved contacts
* 💾 Save contacts in a text file

---

## 🛠️ Technologies Used

* Python 3
* File Handling
* Command Line Interface (CLI)

---

## 📚 Python Concepts Used

* Functions
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Lists
* Dictionaries
* String Manipulation
* File Handling
* Exception Handling

---

## 📂 Project Files

```text
Contact_Book.py      # Main application
Contact book.txt     # Stores contact information
README.md            # Project documentation
```

---

## ▶️ How to Run

1. Download or clone this repository.
2. Make sure Python 3 is installed.
3. Open a terminal or Command Prompt.
4. Run:

```bash
python Contact_Book.py
```
